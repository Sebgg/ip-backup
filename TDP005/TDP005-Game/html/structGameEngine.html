int GameEngine::run ( )

Implementation of GameEngines run function. We create a new PlayState* called current_game. We then use check_win and check_loss to check if the game is won or lost every frame. If it is either we call display_win or display_loss. If the game is not won or lost we call for update_screen and update_gravity and lastly we draw the screen with draw_screen.
