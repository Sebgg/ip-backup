Kompilering av spel:
1. cmake .
2. make spel

För att köra spelet:
./spel

Kontroller:
1. W: Hoppa.
2. A: Gå vänster.
3. S: Droppa från platform.
4. D: Gå höger.
5. Q: Stäng ner spelet, funkar även i vinst-, och förlustskärm.

I latex-mappen finns det en färdigkompilerad refman.pdf, men om man vill
kompilera om den, heter config-filen "Doxyfile"
